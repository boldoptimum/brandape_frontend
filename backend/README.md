# BrandApe Backend Server

This directory contains the `json-server` backend for the BrandApe application.

## Setup

1.  Make sure you are in the `backend` directory.
2.  Install the required dependency:
    ```bash
    npm install
    ```

## Running the Server

To start the API server, run the following command:

```bash
npm start
```

This will start the server on `http://localhost:3001` and watch the `db.json` file for any changes.
